########################      可执行程序说明     ########################

可执行程序为“可执行程序”文件夹中的“MicroChat.exe”文件，监听端口为8010、8030。

一、程序属性
本程序使用Microsoft Visual Studio 2013 Community编写，使用的编程语言是C#，支持的运行环境是Windows 7及以上版本。

二、使用方法
以下使用方法按照使用顺序进行说明。

2.1 更改头像
将鼠标移至登录界面的头像部分，鼠标将变为手指形状，点击后将进入“选择头像”界面。点击任一想要的头像，头像选择成功，“选择头像”界面被关闭，登录界面的头像成功更换为所选头像。

2.2 输入账号密码
程序中的账号密码已预先置为本人学号（2014011541）与实际密码（net2016）。

2.3 登录上线
点击密码框右侧的“→”箭头按钮，程序将向服务器发起请求。若账号密码正确，且校园网环境良好，程序将成功登录上线。

2.4 添加好友
点击主界面右侧的“添加好友”按钮，将弹出“添加好友”界面，输入想要添加好友的学号，点击“查询并添加”，即可向对方发起添加请求。若对方同意，则好友添加成功。点击“刷新好友”按钮即可发现他已经在你的好友列表里了！

2.5 查询好友是否在线
点击“刷新好友”按钮，即可刷新好友的IP地址与状态。若好友在线则会显示“在线”，反之则会显示“离线”

2.6 发起聊天
点击好友列表里任一好友的头像，即可发起与他的一对一聊天。在下方文本框内输入想要发送的文字信息，点击“发送”键即可。

2.7 接受聊天
当好友主动向我发起聊天时，将会弹出提示窗口“您的好友xxx发起会话，开始对话？”。点击“确定”键即可弹出聊天窗口并接收他的会话。

2.8 发送文件
在任一聊天窗口内，点击“发送文件”按钮，在弹出的文件路径浏览框内选择想要发送的文件并确认键，即可向对方发送文件。

2.9 接收文件
当好友主动向我发送文件时，将会弹出提示窗口“您的好友xxx向您发出了文件传输请求，请问是否接收？”。点击“确认”键即可弹出文件保存路径选择窗口并保存文件。

2.10 发起群聊
选中某些好友（点击好友框中的非头像部分），再点击“发起群聊”按钮，即可发起与选中好友的群聊。在下方文本框内输入想要发送的文字信息，点击“发送”键即可。

2.11 接受群聊
当好友主动向我发起群聊时，将会弹出提示窗口“您的好友xxx向您发起了群聊会话，开始对话？”。点击“确认”键即可弹出聊天窗口并接收他的会话。

2.12 退出登录
点击“退出登录”按钮，将会弹出提示“退出登录后将无法接受好友的消息，请问您确定要退出吗？”。点击“确认”后，若服务器回复了成功退出的信息，则退出成功，本账号在服务器的状态为“下线”状态。
